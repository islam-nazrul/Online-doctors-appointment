# Online Doctors Appointment

A modern healthcare appointment booking website built using React.js, Vite, Tailwind CSS, and React Router DOM.

## Features

* Responsive Healthcare UI
* Home Page
* About Page
* Doctors Page
* Appointment Booking Page
* Contact Us Page
* Doctor Cards
* Appointment Form
* Contact Form
* React Router Navigation
* Mobile Responsive Layout
* Reusable Components
* Modern Medical Theme

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router DOM
